# 📚 AI-Powered Study Buddy

An AI-powered study assistant built using Python and Streamlit that helps students learn more effectively through topic explanations, note summarization, quiz generation, flashcards, and learning analytics.

## 🚀 Features

### 📖 Topic Explanation
- Get explanations for study topics.
- Easy-to-understand learning support.

### 📝 Notes Summary
- Generate concise summaries from lengthy notes.
- Save study time and improve revision.

### ❓ Quiz Generator
- Automatically create quiz questions from notes.
- Test understanding and retention.

### 🃏 Flashcards
- Generate flashcards from study material.
- Interactive front/back card format.

### 📊 Analytics Dashboard
- Visual representation of learning activity.
- Track study progress and performance.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/milisrivastav13/AI-Study-Buddy.git
cd AI-Study-Buddy
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
